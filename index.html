06 / Install & Integrations

Fits where you already work.

One CLI install, one config file, native integrations with the 24 tools your engineers use every day. No proxies. No SDK rewrites.

# Install the CLI
npm install -D @tester/cli

# Run your first suite — auto-detects Playwright, Cypress, or Vitest
npx tester run

# Open the dashboard for this run
npx tester open
# Install the CLI
pnpm add -D @tester/cli

# Run with smart-selection — only specs touched by your diff
pnpm tester run --since main

# Watch mode locally
pnpm tester watch
# One-line install — Linux / macOS / WSL
curl -fsSL https://get.tester.io | sh

# Authenticate this machine
tester auth login

# First run
tester run --project myapp
# .github/workflows/test.yml
name: Tests
on: [push, pull_request]
jobs:
  tester:
    runs-on: ubuntu-latest
    steps:
      - uses: tester-io/action@v2
        with:
          token: ${{ secrets.TESTER_TOKEN }}
